# JavaScript 代码规范

一些说明：

- 建议所有人给自己的 IDE 添加上 [editorconfig](http://editorconfig.org) 插件，可以自动应用项目配置的默认基础规则。
- 部分规则来自 [JS风格偏好调查](https://github.com/baixing/Notes/issues/20) 和 [baixing/notes 中的讨论](https://github.com/baixing/notes/issues?utf8=%E2%9C%93&q=is%3Aissue+JavaScript+%E4%BB%A3%E7%A0%81%E9%A3%8E%E6%A0%BC+)
- 新的建议也可以提到 [baixing/notes/issue](https://github.com/baixing/notes/issues) 里

代码规范：

1. 缩进和排版
  - 缩进用制表符（TAB）
  - 排版用空格（SPACE）

1. 行末（声明结束）不用分号
  - 当且仅当行首字符是 `[ ( + - /` 之一时行首使用分号
